# civicsence

CivicSense - AI-Powered Civic Issue Reporting Platform

## Features
- Issue reporting with image upload and AI detection
- Admin dashboard with live detection
- Citizen dashboard
- Multi-language support (EN, HI)
- Voice reporting
- Emergency alert system
- Netlify deployment ready

## Tech Stack
- React 18
- Node.js/Express/MongoDB backend
- TensorFlow.js for AI image detection
- Socket.io real-time
- Leaflet/Mapbox maps

## Deployment
- Frontend: Netlify (connect this repo)
- Backend: Render/ Railway/Vercel

## Quick Start
```
npm install
npm start
```

Backend: `npm start` (port 5000)
Frontend: `npm start` (port 3000)
